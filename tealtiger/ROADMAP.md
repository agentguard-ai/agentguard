## v0.3.0 - Q2 2026

**Focus: Expanded Model Support & Analytics**

### New Features
- [ ] **Gemini Support** - Google's Gemini API integration
- [ ] **Advanced Analytics Dashboard** - Web-based cost and usage analytics
- [ ] **Team Collaboration** - Multi-user support with role-based access

## v0.4.0 - Q3 2026

**Focus: Enterprise Features**

- [ ] **SSO Integration** - SAML/OAuth support
